## Affected Pages/Routes

- `/index.aspx` equivalent new student login page
  Test login success, failure, role-based redirect, stale session, wrong-school access.

- Student home route replacing `student/myinfo.aspx`
  Test current class resolution, empty state, no active lesson, disconnected session recovery.

- Teacher class start page and class session route
  Test choose class, start session, session resume, invalid class assignment, duplicate start protection.

- Teacher radar stream route
  Test SSE connection, reconnect, stale events, teacher switching classes, unauthorized subscription.

- Student heartbeat and help-request routes
  Test heartbeat frequency, expiration, duplicate tabs, offline recovery, help request deduplication.

- Work submission and review routes
  Test draft save, final submit, duplicate submit, teacher review, retry after storage failure.

- Admin migration preview/execute/rollback routes
  Test old Excel template compatibility, schema mismatch, duplicate legacy IDs, partial failures, rollback behavior.

- Teacher AI suggestion routes
  Test suggestion generation, timeout, invalid output filtering, edit-then-accept flow, rejection flow.

## Key Interactions to Verify

- Teacher logs in, starts class, sees radar populate with real student states.
- Student logs in, enters class, heartbeat becomes visible to teacher.
- Student asks for help and teacher sees it without page refresh.
- Admin uploads old Excel, previews diff, executes import, handles partial failure, retries safely.
- Teacher requests AI lesson suggestion, edits draft, saves approved version.
- Teacher requests AI feedback draft for student work, reviews and publishes manually.

## Edge Cases

- Wrong password, expired session, role mismatch.
- Student opens same class in two tabs.
- Teacher changes selected class while radar stream is open.
- Student network drops during class and reconnects after timeout.
- Import file has missing columns, extra columns, wrong data types, duplicate IDs.
- Import batch partially succeeds, then rollback is triggered.
- AI provider timeout, malformed output, empty output, model unavailable.
- Object storage write fails during submission or migration.

## Critical Paths

- Student login -> active class -> heartbeat -> work submission.
- Teacher login -> start class -> watch radar -> review help request -> review work.
- Admin login -> import preview -> execute migration -> inspect errors -> rollback if needed.
- Teacher request AI draft -> review/edit -> accept and publish.
